## 🚀 Getting Started

### Prerequisites

* Node.js (v18 or above)
* npm

---

### Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/sandeep-portfolio.git
```

Navigate into the project

```bash
cd sandeep-portfolio
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
